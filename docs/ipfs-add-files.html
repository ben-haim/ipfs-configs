IPFS files API
IPFS-API-HOST : https://ipfs1-api.horizontalsystems.xyz
IPFS-HOST : https://ipfs1.horizontalsystems.xyz
1. Add files to IPFS

To add files to IPFS first of all generate Uniq HASH from 12 words wallet secret. Let it be "H1". Then add "H1" as "arg" parameter to Get request URL.


Input:
URL GET:{IPFS-API-HOST}/api/v0/write?
No Name Details
1 arg Path + File name to be added.
2 parents Create parent folders
3 create Create file if not exists
Output:
HTTP Status : 200
2. Get Hash of the IPFS folder
To read files from IPFS you should get the HASH of the parent folder. (H1)
Input:
URL: {IPFS-API-HOST}/api/v0/files/stat?
No Name Details
1 arg Path of the files. (H1 generated from wallet secret)
Output:
3. Read data from IPFS using HASH
To read files from IPFS you should get the HASH of the parent folder. (H1)
Input:
URL: {IPFS-HOST}/ipfs/{HASH_OF_THE_FOLDER}/
No Name Details
1 HASH_OF_THE_FOLDER Hash of the folder get from API: files/stat
Output:
HTTP Status: 200
